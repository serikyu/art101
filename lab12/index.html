Lab 12: Conditionals

Challenge

In this assignment, we were tasked with creating a JavaScript function that would take your name and put you into a Hogwarts house based on the length of your name.

Problems

I wanted to figure out a different way to sort out the names rather than by just length, but for the sake of time, I just followed the assignment as is.

Reflection

I understand how conditionals work, and how I can utilize else/else if statements. I realized that hierarchies in the code for conditionals should exist to make things cleaner, but luckily I didn't run into that problem in this particular assignment.
